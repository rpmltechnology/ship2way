Product Import


If you'd like to import product information from a spreadsheet or other datasource, you can provide your data in our standard CSV file format. As products are imported we will create products that do not already exist and optionally update existing products.

Download a sample CSV file

1. Select a File to Import


2. Select Options

As products are imported, we will attempt to identify whether they already exist. You can select which field uniquely identifies a product.

You can choose to update products if they already exist or to only import new products.
